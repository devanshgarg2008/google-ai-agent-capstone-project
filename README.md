# google-ai-agent-capstone-project
This repository contains "My Capstone Project" developed as part of the "Google AI Agent Workshop" learning program.  Capstone project name : Customer-helper-agent.  
This project focuses on understanding the customer problem "What problem are facing the customer" and implementing AI agent concepts such as  decision-making, and give the response to the customer After analysing the customer website page/id and the customer problem with the help of intelligent workflow execution.
